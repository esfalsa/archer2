Archer is a user-friendly endorsement tool provided by The South Pacific. We want to make endorsing easy and quick, so that you can endorse the world! There are two versions available:

Interactive

The interactive version is intended for desktop use. Since your localid is prefilled, your endorsement is given automatically when you press an endorsement button.

Static

The static version is intended for mobile use. It creates the same buttons as the interactive version, but you will need to manually give your endorsement.

Both versions will open up a new tab when clicking an endorsement button. Subsequent clicks will use the same tab, as long as it remains open. On mobile platforms, you'll need to flip back and forth. For desktop, we recommend putting the new tab that gets created side-by-side with Archer, so you don't need to flip back and forth.
