Archer is a user-friendly endorsement tool provided by The South Pacific. We want to make endorsing easy and quick, so that you can endorse the world! There are two versions available:

Interactive

The interactive version is intended for desktop use. Since your localid is prefilled, your endorsement is given automatically when you press an endorsement button. This version requires frame support and third-party cookies (required so that you can be logged into NS inside the frame).

Static

The static version is intended for mobile use. It creates the same buttons as the interactive version, but will open a new window and you will need to manually give your endorsement.

Privacy

Archer doesn't connect to NationStates in any way that can read what you enter or do within the NationStates frame generated by the interactive version. The only data Archer collects and stores is the list of nations generated, which allows you to leave and come back to endorse in your own time. This data can be cleared by using the "Clear Arhcer Data" button at the bottom of any page.
